## Project Overview
This project is an **Autonomous Agronomy Assistant** designed to help farmers and gardeners plan their crop cycles. Unlike standard chatbots, this agent:
1.  **Autonomous Research**: actively searches the web for verified data if it doesn't know a crop (e.g., "Ghost Peppers").
2.  **Strict Security**: Enforces a strict whitelist, only learning from trusted gov/edu sites (FAO, USDA, Cornell).
3.  **Structured Planning**: Generates a strictly formatted **JSON Timetable** linking input names (Ingredients) to specific dates (Timetable).
4.  **Local Privacy**: Runs 100% locally using `Mistral-7B` via `llama.cpp`.

## UI Visualization
The project features a **Streamlit Web Interface**.
*   **See it in action**: Open `frontend_snapshot.html` in your browser to see a captured state of the interface.
*   **Clean Design**: Features a chat interface where complex JSON is rendered into readable tables.

## System Capabilities
*   **Autonomous Data Collection**: `duckduckgo-search` + `ScraperTool`.
*   **Intelligent Synthesis**: `Mistral-7B` (Local) with Schema Fallback.
*   **Robustness**: Error handling for JSON validation failures ensures data delivery.

## Limitations & Improvements
*   **Inference Speed**: Local CPU inference takes ~30-60s.
*   **Future**: Plan to upgrade to `Mixtral-8x7B` and add PDF export functionality.
